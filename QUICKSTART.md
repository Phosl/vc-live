# Quickstart

```bash
npm install
cp .env.example .env
```

Apri `.env`, inserisci la tua API key:

```env
OPENAI_API_KEY=sk-proj-...
```

Avvia:

```bash
npm run dev
```

Apri:

```text
http://127.0.0.1:5173
```

Poi:

1. **Condividi schermo / VS Code**
2. scegli VS Code;
3. **Disegna rettangolo** sulla chat;
4. **Leggi una volta**;
5. **Avvia live**.
